typeSctipt基础笔记

Posted on 2018-06-06 | In typescript

元组 Tuple

元组是js中不存在的数据类型，它表示一个已知元素数量和类型的数组，但是各个元素的类型不必相同

枚举 enum

为一组数值赋予有好的名字

因为没有学过c#所以一开始理解enum的作用有点不理解，然后观察它的形式，发现
枚举类型就是一个没有只有key的对象，但是枚举类型上的key可以随意赋值。

可以手动进行全部赋值

枚举的方便之处就是可以随意给名字进行赋值，同时也可以通过值查找名字

any

any指定后，可以定义任何数据类型

void

表示没有任何数据类型

Null/Undefined

它们两个是所有类型的子类，也就是说可以赋值给所有的类型

Never

never表示那些用不存在的值的类型，例如：没有返回值的函数或者是箭头函数的返回值类型，never是任何类型的子类型，可以赋值给任何类型

断言

跟js的类型检查是一样的，分为两种语法，但是因为jsx中只能用 as 所以我打算只用as
